# 1. 버튼 온클릭 처리 문제

```kotlin
Icon(
    modifier = Modifier
        .padding(16.dp)
        .runIf(startIconData.onClick != null) {
            clickable {
                startIconData.onClick!!.invoke()
            }
        },
    painter = painterResource(startIconData.resId),
    contentDescription = null,
    tint = Color.Unspecified
)
```
